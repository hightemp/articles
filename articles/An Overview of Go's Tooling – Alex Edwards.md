–í –ø—Ä–∏–≤–µ–¥–µ–Ω–Ω–æ–º –≤—ã—à–µ –ø—Ä–∏–º–µ—Ä–µ —è –¥–≤–∞–∂–¥—ã –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–ª —Ñ–ª–∞–≥ `-m`, —á—Ç–æ–±—ã —É–∫–∞–∑–∞—Ç—å, —á—Ç–æ —è —Ö–æ—á—É –≤—ã–≤–µ—Å—Ç–∏ –∏–Ω—Ñ–æ—Ä–º–∞—Ü–∏—é –æ —Ä–µ—à–µ–Ω–∏—è—Ö –Ω–∞ –¥–≤—É—Ö —É—Ä–æ–≤–Ω—è—Ö. –í—ã –º–æ–∂–µ—Ç–µ –ø–æ–ª—É—á–∏—Ç—å –±–æ–ª–µ–µ –ø—Ä–æ—Å—Ç–æ–π —Ä–µ–∑—É–ª—å—Ç–∞—Ç, –∏—Å–ø–æ–ª—å–∑—É—è —Ç–æ–ª—å–∫–æ –æ–¥–∏–Ω —Ñ–ª–∞–≥.

–ö—Ä–æ–º–µ —Ç–æ–≥–æ, –Ω–∞—á–∏–Ω–∞—è —Å Go 1.10, —Ñ–ª–∞–≥–∏ –∫–æ–º–ø–∏–ª—è—Ç–æ—Ä–∞ –ø—Ä–∏–º–µ–Ω—è—é—Ç—Å—è —Ç–æ–ª—å–∫–æ –∫ –∫–æ–Ω–∫—Ä–µ—Ç–Ω—ã–º –ø–∞–∫–µ—Ç–∞–º, –ø–µ—Ä–µ–¥–∞–Ω–Ω—ã–º –≤ `go build` ‚Äî –≤ –ø—Ä–∏–≤–µ–¥—ë–Ω–Ω–æ–º –≤—ã—à–µ –ø—Ä–∏–º–µ—Ä–µ —ç—Ç–æ –ø–∞–∫–µ—Ç –≤ —Ç–µ–∫—É—â–µ–º –∫–∞—Ç–∞–ª–æ–≥–µ (–æ–±–æ–∑–Ω–∞—á–∞–µ—Ç—Å—è `.`). –ï—Å–ª–∏ –≤—ã —Ö–æ—Ç–∏—Ç–µ –≤—ã–≤–µ—Å—Ç–∏ —Ä–µ—à–µ–Ω–∏—è –ø–æ –æ–ø—Ç–∏–º–∏–∑–∞—Ü–∏–∏ –¥–ª—è –≤—Å–µ—Ö –ø–∞–∫–µ—Ç–æ–≤, –≤–∫–ª—é—á–∞—è –∑–∞–≤–∏—Å–∏–º–æ—Å—Ç–∏, –∏—Å–ø–æ–ª—å–∑—É–π—Ç–µ –≤–º–µ—Å—Ç–æ —ç—Ç–æ–≥–æ –∫–æ–º–∞–Ω–¥—É:

–ù–∞—á–∏–Ω–∞—è —Å Go 1.11, –≤–∞–º –¥–æ–ª–∂–Ω–æ –±—ã—Ç—å [–ª–µ–≥—á–µ –æ—Ç–ª–∞–∂–∏–≤–∞—Ç—å –æ–ø—Ç–∏–º–∏–∑–∏—Ä–æ–≤–∞–Ω–Ω—ã–µ –¥–≤–æ–∏—á–Ω—ã–µ —Ñ–∞–π–ª—ã](https://golang.org/doc/go1.11#debugging), —á–µ–º —Ä–∞–Ω—å—à–µ. –û–¥–Ω–∞–∫–æ –≤—ã –ø–æ-–ø—Ä–µ–∂–Ω–µ–º—É –º–æ–∂–µ—Ç–µ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å —Ñ–ª–∞–≥–∏ `-N` –¥–ª—è –æ—Ç–∫–ª—é—á–µ–Ω–∏—è –æ–ø—Ç–∏–º–∏–∑–∞—Ü–∏–∏ –∏ `-l` –¥–ª—è –æ—Ç–∫–ª—é—á–µ–Ω–∏—è –≤—Å—Ç—Ä–∞–∏–≤–∞–Ω–∏—è, –µ—Å–ª–∏ –≤–∞–º —ç—Ç–æ –Ω—É–∂–Ω–æ. –ù–∞–ø—Ä–∏–º–µ—Ä:

–í—ã –º–æ–∂–µ—Ç–µ –ø—Ä–æ—Å–º–æ—Ç—Ä–µ—Ç—å —Å–ø–∏—Å–æ–∫ –¥–æ—Å—Ç—É–ø–Ω—ã—Ö —Ñ–ª–∞–≥–æ–≤ –∫–æ–º–ø–æ–Ω–æ–≤—â–∏–∫–∞, –≤—ã–ø–æ–ª–Ω–∏–≤ –∫–æ–º–∞–Ω–¥—É:

`$ go tool link -help` 

–í–µ—Ä–æ—è—Ç–Ω–æ, —Å–∞–º—ã–π –∏–∑–≤–µ—Å—Ç–Ω—ã–π –∏–∑ –Ω–∏—Ö ‚Äî —ç—Ç–æ —Ñ–ª–∞–≥ `-X`, –∫–æ—Ç–æ—Ä—ã–π –ø–æ–∑–≤–æ–ª—è–µ—Ç ¬´–ø—Ä–∏—Å–≤–æ–∏—Ç—å¬ª (—Å—Ç—Ä–æ–∫–æ–≤–æ–µ) –∑–Ω–∞—á–µ–Ω–∏–µ –æ–ø—Ä–µ–¥–µ–ª—ë–Ω–Ω–æ–π –ø–µ—Ä–µ–º–µ–Ω–Ω–æ–π –≤ –≤–∞—à–µ–º –ø—Ä–∏–ª–æ–∂–µ–Ω–∏–∏. –û–±—ã—á–Ω–æ —ç—Ç–æ –∏—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è –¥–ª—è [–¥–æ–±–∞–≤–ª–µ–Ω–∏—è –Ω–æ–º–µ—Ä–∞ –≤–µ—Ä—Å–∏–∏ –∏–ª–∏ —Ö—ç—à–∞ –∫–æ–º–º–∏—Ç–∞](https://blog.alexellis.io/inject-build-time-vars-golang/). –ù–∞–ø—Ä–∏–º–µ—Ä:

`$ go build -ldflags="-X main.version=1.2.3" -o=/tmp/foo .` 

–î–æ–ø–æ–ª–Ω–∏—Ç–µ–ª—å–Ω—É—é –∏–Ω—Ñ–æ—Ä–º–∞—Ü–∏—é –æ —Ñ–ª–∞–≥–µ `-X` –∏ –ø—Ä–∏–º–µ—Ä—ã –∫–æ–¥–∞ —Å–º. –≤ [—ç—Ç–æ–º –≤–æ–ø—Ä–æ—Å–µ –Ω–∞ StackOverflow](https://stackoverflow.com/questions/11354518/golang-application-auto-build-versioning), –∞ —Ç–∞–∫–∂–µ –≤ —ç—Ç–æ–º –∏ [—ç—Ç–æ–º –ø–æ—Å—Ç–∞—Ö](https://blog.alexellis.io/inject-build-time-vars-golang/).

–í–∞–º —Ç–∞–∫–∂–µ –º–æ–∂–µ—Ç –±—ã—Ç—å –∏–Ω—Ç–µ—Ä–µ—Å–Ω–æ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å —Ñ–ª–∞–≥–∏ `-s` –∏ `-w` –¥–ª—è —É–¥–∞–ª–µ–Ω–∏—è –æ—Ç–ª–∞–¥–æ—á–Ω–æ–π –∏–Ω—Ñ–æ—Ä–º–∞—Ü–∏–∏ –∏–∑ –¥–≤–æ–∏—á–Ω–æ–≥–æ —Ñ–∞–π–ª–∞. –û–±—ã—á–Ω–æ —ç—Ç–æ –ø–æ–∑–≤–æ–ª—è–µ—Ç —É–º–µ–Ω—å—à–∏—Ç—å –∫–æ–Ω–µ—á–Ω—ã–π —Ä–∞–∑–º–µ—Ä –ø—Ä–∏–º–µ—Ä–Ω–æ –Ω–∞ 25%. –ù–∞–ø—Ä–∏–º–µ—Ä:

`$ go build -ldflags="-s -w" -o=/tmp/foo .  # Strip debug information from the binary` 

–î–∏–∞–≥–Ω–æ—Å—Ç–∏–∫–∞ –ü—Ä–æ–±–ª–µ–º –∏ –≤–Ω–µ—Å–µ–Ω–∏–µ –û–ø—Ç–∏–º–∏–∑–∞—Ü–∏–π

### –ó–∞–ø—É—Å–∫ –∏ —Å—Ä–∞–≤–Ω–µ–Ω–∏–µ –ë–µ–Ω—á–º–∞—Ä–∫–æ–≤

–ü—Ä–∏—è—Ç–Ω–æ–π –æ—Å–æ–±–µ–Ω–Ω–æ—Å—Ç—å—é Go —è–≤–ª—è–µ—Ç—Å—è —Ç–æ, —á—Ç–æ –æ–Ω –ø–æ–∑–≤–æ–ª—è–µ—Ç –ª–µ–≥–∫–æ –ø—Ä–æ–≤–æ–¥–∏—Ç—å —Ç–µ—Å—Ç–∏—Ä–æ–≤–∞–Ω–∏–µ –ø—Ä–æ–∏–∑–≤–æ–¥–∏—Ç–µ–ª—å–Ω–æ—Å—Ç–∏ –≤–∞—à–µ–≥–æ –∫–æ–¥–∞. –ï—Å–ª–∏ –≤—ã –Ω–µ –∑–Ω–∞–∫–æ–º—ã —Å –æ–±—â–∏–º –ø—Ä–æ—Ü–µ—Å—Å–æ–º –Ω–∞–ø–∏—Å–∞–Ω–∏—è —Ç–µ—Å—Ç–æ–≤ –ø—Ä–æ–∏–∑–≤–æ–¥–∏—Ç–µ–ª—å–Ω–æ—Å—Ç–∏, –µ—Å—Ç—å —Ö–æ—Ä–æ—à–∏–µ —Ä—É–∫–æ–≤–æ–¥—Å—Ç–≤–∞ [–∑–¥–µ—Å—å](https://dave.cheney.net/2013/06/30/how-to-write-benchmarks-in-go) –∏ [–∑–¥–µ—Å—å](https://dave.cheney.net/2013/06/30/how-to-write-benchmarks-in-go).

–ß—Ç–æ–±—ã –∑–∞–ø—É—Å—Ç–∏—Ç—å —Ç–µ—Å—Ç—ã, –≤–∞–º –Ω—É–∂–Ω–æ –±—É–¥–µ—Ç –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å –∏–Ω—Å—Ç—Ä—É–º–µ–Ω—Ç `go test` —Å —Ñ–ª–∞–≥–æ–º `-bench` –∏ —Ä–µ–≥—É–ª—è—Ä–Ω—ã–º –≤—ã—Ä–∞–∂–µ–Ω–∏–µ–º, –∫–æ—Ç–æ—Ä–æ–µ —Å–æ–æ—Ç–≤–µ—Ç—Å—Ç–≤—É–µ—Ç —Ç–µ—Å—Ç–∞–º, –∫–æ—Ç–æ—Ä—ã–µ –≤—ã —Ö–æ—Ç–∏—Ç–µ –≤—ã–ø–æ–ª–Ω–∏—Ç—å. –ù–∞–ø—Ä–∏–º–µ—Ä:

`$ go test -bench=. ./...                        # Run all benchmarks and tests
$ go test -run=^$ -bench=. ./...                # Run all benchmarks (and no tests)

–Ø –ø–æ—á—Ç–∏ –≤—Å–µ–≥–¥–∞ –∑–∞–ø—É—Å–∫–∞—é —Ç–µ—Å—Ç—ã —Å —Ñ–ª–∞–≥–æ–º `-benchmem`, –∫–æ—Ç–æ—Ä—ã–π –∑–∞—Å—Ç–∞–≤–ª—è–µ—Ç –≤–∫–ª—é—á–∞—Ç—å –≤ –≤—ã–≤–æ–¥ —Å—Ç–∞—Ç–∏—Å—Ç–∏–∫—É —Ä–∞—Å–ø—Ä–µ–¥–µ–ª–µ–Ω–∏—è –ø–∞–º—è—Ç–∏.

`$  go test -bench=. -benchmem ./...` 

–ü–æ —É–º–æ–ª—á–∞–Ω–∏—é –∫–∞–∂–¥—ã–π —Ç–µ—Å—Ç –ø—Ä–æ–∏–∑–≤–æ–¥–∏—Ç–µ–ª—å–Ω–æ—Å—Ç–∏ –±—É–¥–µ—Ç –≤—ã–ø–æ–ª–Ω—è—Ç—å—Å—è _–º–∏–Ω–∏–º—É–º_ 1 —Å–µ–∫—É–Ω–¥—É —Ç–æ–ª—å–∫–æ –æ–¥–∏–Ω —Ä–∞–∑. –í—ã –º–æ–∂–µ—Ç–µ –∏–∑–º–µ–Ω–∏—Ç—å —ç—Ç–æ —Å –ø–æ–º–æ—â—å—é —Ñ–ª–∞–≥–æ–≤ `-benchtime` –∏ `-count`:

`$ go test -bench=. -benchtime=5s ./...       # Run each benchmark test for at least 5 seconds

$ go test -bench=. -count=3 ./...            # Repeat each benchmark test 3 times over` 

–ï—Å–ª–∏ –∫–æ–¥, –∫–æ—Ç–æ—Ä—ã–π –≤—ã —Ç–µ—Å—Ç–∏—Ä—É–µ—Ç–µ, –∏—Å–ø–æ–ª—å–∑—É–µ—Ç –ø–∞—Ä–∞–ª–ª–µ–ª–∏–∑–º, –≤—ã –º–æ–∂–µ—Ç–µ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å —Ñ–ª–∞–≥ `-cpu` –¥–ª—è –ø—Ä–æ—Å–º–æ—Ç—Ä–∞ –≤–ª–∏—è–Ω–∏—è –Ω–∞ –ø—Ä–æ–∏–∑–≤–æ–¥–∏—Ç–µ–ª—å–Ω–æ—Å—Ç—å –∏–∑–º–µ–Ω–µ–Ω–∏—è –∑–Ω–∞—á–µ–Ω–∏—è `GOMAXPROCS` (–ø–æ —Å—É—Ç–∏, –∫–æ–ª–∏—á–µ—Å—Ç–≤–∞ –ø–æ—Ç–æ–∫–æ–≤ –û–°, –∫–æ—Ç–æ—Ä—ã–µ –º–æ–≥—É—Ç –æ–¥–Ω–æ–≤—Ä–µ–º–µ–Ω–Ω–æ –≤—ã–ø–æ–ª–Ω—è—Ç—å –≤–∞—à –∫–æ–¥ Go). –ù–∞–ø—Ä–∏–º–µ—Ä, —á—Ç–æ–±—ã –∑–∞–ø—É—Å—Ç–∏—Ç—å —Ç–µ—Å—Ç—ã —Å `GOMAXPROCS` —Ä–∞–≤–Ω—ã–º 1, 4 –∏ 8:

`$ go test -bench=. -cpu=1,4,8 ./...` 

–ß—Ç–æ–±—ã —Å—Ä–∞–≤–Ω–∏—Ç—å –∏–∑–º–µ–Ω–µ–Ω–∏—è –º–µ–∂–¥—É –±–µ–Ω—á–º–∞—Ä–∫–∞–º–∏, –≤—ã –º–æ–∂–µ—Ç–µ –∑–∞—Ö–æ—Ç–µ—Ç—å –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å –∏–Ω—Å—Ç—Ä—É–º–µ–Ω—Ç [benchcmp](https://godoc.org/golang.org/x/tools/cmd/benchcmp). –≠—Ç–æ –Ω–µ —á–∞—Å—Ç—å —Å—Ç–∞–Ω–¥–∞—Ä—Ç–Ω–æ–π `go` –∫–æ–º–∞–Ω–¥—ã, –ø–æ—ç—Ç–æ–º—É –≤–∞–º –Ω—É–∂–Ω–æ –±—É–¥–µ—Ç —É—Å—Ç–∞–Ω–æ–≤–∏—Ç—å –µ–µ —Å–ª–µ–¥—É—é—â–∏–º –æ–±—Ä–∞–∑–æ–º:

–ó–∞—Ç–µ–º –≤—ã –º–æ–∂–µ—Ç–µ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å –µ–≥–æ —Å–ª–µ–¥—É—é—â–∏–º –æ–±—Ä–∞–∑–æ–º:

### –ü—Ä–æ—Ñ–∏–ª–∏—Ä–æ–≤–∞–Ω–∏–µ –∏ –æ—Ç—Å–ª–µ–∂–∏–≤–∞–Ω–∏–µ

Go –ø–æ–∑–≤–æ–ª—è–µ—Ç —Å–æ–∑–¥–∞–≤–∞—Ç—å –¥–∏–∞–≥–Ω–æ—Å—Ç–∏—á–µ—Å–∫–∏–µ _–ø—Ä–æ—Ñ–∏–ª–∏_ –¥–ª—è –æ—Ç—Å–ª–µ–∂–∏–≤–∞–Ω–∏—è –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏—è –ø—Ä–æ—Ü–µ—Å—Å–æ—Ä–∞, –ø–∞–º—è—Ç–∏, –±–ª–æ–∫–∏—Ä–æ–≤–∫–∏ –≥–æ—Ä—É—Ç–∏–Ω –∏ –∫–æ–Ω–∫—É—Ä–µ–Ω—Ü–∏–∏ –∑–∞ –º—å—é—Ç–µ–∫—Å—ã. –° –∏—Ö –ø–æ–º–æ—â—å—é –º–æ–∂–Ω–æ –∫–æ–ø–Ω—É—Ç—å –≥–ª—É–±–∂–µ –∏ —É–≤–∏–¥–µ—Ç—å, –∫–∞–∫ –∏–º–µ–Ω–Ω–æ –≤–∞—à–µ –ø—Ä–∏–ª–æ–∂–µ–Ω–∏–µ –∏—Å–ø–æ–ª—å–∑—É–µ—Ç (–∏–ª–∏ –æ–∂–∏–¥–∞–µ—Ç) —Ä–µ—Å—É—Ä—Å—ã.

–°—É—â–µ—Å—Ç–≤—É–µ—Ç —Ç—Ä–∏ —Å–ø–æ—Å–æ–±–∞ —Å–æ–∑–¥–∞–Ω–∏—è –ø—Ä–æ—Ñ–∏–ª–µ–π:

*   –î–ª—è –¥—Ä—É–≥–∏—Ö —Ç–∏–ø–æ–≤ –ø—Ä–∏–ª–æ–∂–µ–Ω–∏–π –≤—ã –º–æ–∂–µ—Ç–µ –ø—Ä–æ—Ñ–∏–ª–∏—Ä–æ–≤–∞—Ç—å —Ä–∞–±–æ—Ç–∞—é—â–µ–µ –ø—Ä–∏–ª–æ–∂–µ–Ω–∏–µ —Å –ø–æ–º–æ—â—å—é —Ñ—É–Ω–∫—Ü–∏–π `pprof.StartCPUProfile()` –∏ `pprof.WriteHeapProfile()`. –ü—Ä–∏–º–µ—Ä—ã –∫–æ–¥–∞ —Å–º. –≤ –¥–æ–∫—É–º–µ–Ω—Ç–∞—Ü–∏–∏ [`runtime/pprof`](https://pkg.go.dev/runtime/pprof/)
